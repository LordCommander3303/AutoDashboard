# 🤖 AI Data Architect

A minimalist, high-performance executive dashboard that transforms raw CSV data into actionable business insights using **Llama 3** and **Flask**.

## 🚀 Key Features
*   **Natural Language Querying:** Ask complex questions about your data (e.g., "Where am I losing money?") and get immediate visual answers.
*   **Intelligent Auto-Visualization:** Automatically selects the most impactful chart types (Bar, Line, etc.) based on data metadata.
*   **Executive Zen UI:** A clean, distraction-free interface designed for decision-makers.

## 🛠️ Tech Stack
*   **Backend:** Python / Flask
*   **AI Engine:** Groq SDK (Llama 3 8B)
*   **Frontend:** Tailwind CSS / Plotly.js
*   **Data Processing:** Pandas

## 📖 How to Use
1.  Upload any standard CSV file.
2.  Enter a strategic directive or leave it blank for a general overview.
3.  Analyze the AI-generated trends and rationales instantly.
